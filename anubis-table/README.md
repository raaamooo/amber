# Anubis' Table — A Pharaoh's Menu

Welcome to **Anubis' Table**, a production-ready, mobile-first restaurant menu website themed around Ancient Egyptian mythology with a modern twist. Step into a pharaoh's tomb serving modern Egyptian cuisine.

## Features
- **Mythological Theme**: "Cinzel" fonts, floating hieroglyphs, gold accents, and a custom Ankh cursor.
- **Mobile-Optimized**: Designed mobile-first for small screens (360px+) scaling up smoothly.
- **Theme Switcher**: "Tomb at Midnight" (Dark) and "Desert at Noon" (Light) mode toggle with `localStorage` persistence.
- **Dynamic Menu**: Populated via JavaScript, featuring Feasts, Nectars, and Sweets.
- **Interactive Game**: "The Trial of Anubis" quiz algorithm that matches user preferences to a menu item.
- **Animations**: Typing text effect for Anubis, scroll-revealing papyrus cards, golden hover glows.

## Setup & Local Development
1. Clone the repository.
2. Open `index.html` in your web browser. 
3. Images are ready to be placed in `assets/images/`.

## Deployment Guide (GitHub Pages)
1. Initialize a git repository in this folder: `git init`
2. Add files: `git add .`
3. Commit: `git commit -m "Initial commit"`
4. Create a new repository on GitHub and link it: `git remote add origin <your-repo-url>`
5. Push your code: `git push -u origin main`
6. On GitHub, navigate to **Settings > Pages**.
7. Under "Source", select the `main` branch and `/ (root)` folder.
8. Save. Your site will be published at `https://<your-username>.github.io/<repository-name>/`.
